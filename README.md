# File_Transfer


download 
----------
```
git clone https://github.com/0UN390/File_Transfer 
cd File_Transfer
```


usage 
----------
```
send:
sudo chmod 775 send  
./send "file" "host ip" 

recive:
sudo chmod 775 recive 
sudo ./recive
```
